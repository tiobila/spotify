Alguma dúvida?

Como faço para criar uma playlist? Como faço para ativar o modo de Economia de dados? Só é possível ouvir música em ordem aleatória? Onde posso encontrar podcasts?

As playlists são uma ótima maneira de criar coleções de músicas, seja para só você ouvir ou para compartilhar. Para criar uma: 1. Clique na “Sua Biblioteca”. 2. Clique em CRIAR. 3. Dê um nome para a playlist. 4. Comece a incluir algumas músicas (enquanto recomendamos outras).

1. Clique em Início. 2. Clique em Configurações. 3. Clique em Economia de dados. 4. Ative a “Economia de dados”.

As playlists com o ícone de ordem aleatória tocam dessa forma. Nas playlists que não têm esse ícone, você pode clicar em qualquer música para ouvir.

Clique em Buscar. Em Navegar por todas as seções, clique em Podcasts.
